# Weather-ETL-in-Pentaho
Weather ETL Pipeline using Pentaho
<ul>
<li>10 cities were filtered randomly from the cities.json file.</li><br>
<li>OpenWeather API was used to extract the weather data for the cities.</li
</ul><br><br>
